## <span style="color:#9388E8">Introduce 배우자</span>💜
### <span style="color:#9388E8">배우자</span>는 K-Contents(K-POP, K-MOVIE, K-DRAMA)를 이용한 한국어 학습 앱 입니다!
- __<span style="color:#9388E8">`K-POP`</span> 뮤직 비디오를 보면서 가사의 <span style="color:#9388E8">`단어`</span>를 학습합니다!__
- __<span style="color:#9388E8">`K-MOVIE`, `K-DRAMA`</span>를 보면서 <span style="color:#9388E8">`회화 표현`</span>을 학습합니다!__
### 학습 도구는 <span style="color:#9388E8">한국인 성우 음성, 자신의 발화 녹음 및 듣기, 발화 레벨 평가</span> 기능을 제공합니다!
- __한국인 성우 음성 : 가사, 회화 표현을 <span style="color:#9388E8">`한국인 성우의 음성`</span> 으로 들어보며 정확한 발화 이해__
- __자신의 발화 녹음 및 듣기 : 성우의 음성과 <span style="color:#9388E8">`자신의 음성`</span>을 비교하며 학습__
- __발화 레벨 평가 : 자신의 발화를 <span style="color:#9388E8">`AI`</span>가 분석해 <span style="color:#9388E8">`발화 수준`</span>과 <span style="color:#9388E8">`pitch`</span> 그래프를 통해 확인__
### 학습한 내용은 <span style="color:#9388E8">퀴즈</span>를 통해 <span style="color:#9388E8">복습</span>할 수 있습니다!
- __내가 얼마나 학습했는 지 <span style="color:#9388E8">`학습률`</span>을 제공합니다!__
- __복습을 도와주는 <span style="color:#9388E8">`푸쉬 알림`</span> 기능__
